# Aura Tiles - Premium Tiles Collection

A professional and responsive web application designed for a premier tile shop in Bangladesh. This platform showcases an extensive collection of ceramic, marble, and granite tiles with a focus on minimalist aesthetics, high-performance filtering, and secure authentication.

---

## 🚀 Purpose
The **Aura Tiles** project was developed to provide a seamless digital catalog experience for customers. It addresses the need for a fast, responsive interface where users can browse, search by title, and filter various tile types without unnecessary page reloads, ensuring a modern shopping experience.

---

## ✨ Key Features
*   **Dynamic Search:** Instant filtering of tiles by title directly in the browser for a fast user experience.
*   **Secure Authentication:** User login and session management powered by Better Auth with a MongoDB adapter.
*   **Modern UI Components:** Built with HeroUI and custom animations using Animate.css and React Fast Marquee.
*   **Interactive Galleries:** Touch-friendly image sliders implemented with Swiper.
*   **Responsive Design:** Fully optimized across Mobile, Tablet, and Desktop breakpoints using Tailwind CSS v4.
*   **Form Handling & Validation:** Robust form state management using React Hook Form.
*   **Toast Notifications:** Elegant user feedback provided by Sonner.

---

## 🛠️ Tech Stack & Packages

### Core Frameworks
*   **Next.js (v16.2.4)**: React framework for production with App Router.
*   **React (v19.2.4)**: Library for building user interfaces.


### UI & Styling
*   **Tailwind CSS (v4)**: Utility-first CSS framework for responsive design.
*   **HeroUI (@heroui/react v3.0.3)**: Premium UI library for accessible components.
*   **Swiper**: Modern mobile touch slider for image galleries.
*   **Lucide React & React Icons**: Clean, consistent iconography.
*   **Gravity UI Icons**: Extended icon library.
*   **Animate.css**: Plug-and-play CSS animations.

### Database & Authentication
*   **MongoDB (v7.2.0)**: NoSQL database for flexible data storage.
*   **Better Auth (v1.6.9)**: Authentication framework with `@better-auth/mongo-adapter`.
*   **JSON Server**: Local mock REST API for development testing.
